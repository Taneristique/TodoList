# todoList

This project creates a simple daily task list and caches it inside of tasks.txt, feel free to enjoy or modify it.

## Getting Started 

git clone "https://github.com/Taneristique/TodoList.git"
cd TodoList 
gcc main.c -o main.sh
chmod +w+r ./main.sh
./main.sh
